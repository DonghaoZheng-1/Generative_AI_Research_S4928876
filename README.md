What does git clone do; what does github fork do. Differences between git clone and github fork

git clone: Git clone copies a remote repository from GitHub to your local computer. It downloads the files, commit history, and branches so you can work on the project locally.

github fork:GitHub fork creates a copy of another person’s repository under your own GitHub account. This allows you to have your own online version of the project without changing the original repository.

differences:The main difference is that git clone makes a local copy on your computer, while GitHub fork makes an online copy in your GitHub account. A common workflow is to first fork a repository on GitHub and then clone that fork to your local machine.
